# Wordle Archive

A complete archive of every [Wordle](https://www.nytimes.com/games/wordle/index.html) answer, from the very first puzzle on June 19, 2021 to the present (including some future words!).

## Structure

```
wordle_words/
├── 2021/
│   ├── June/
│   │   └── 19.txt   ← CIGAR (the first ever Wordle word!)
│   └── ...
├── 2022/
├── 2023/
├── 2024/
├── 2025/
└── 2026/

```

Each `.txt` file contains a single word in uppercase. 

## Notes

- The archive includes some future dates that NYT pre-loads in advance
- This repo is for archival and educational purposes only
